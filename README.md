# DigBot web-crawler

CCTLD focused, recursive crawler built with Scrapy.

# Requirements

* Python (2.7)
* Scrapy (0.24.4)
* scrapy-redis (0.5.2)
* psycopg2 (2.5.4)
* SQLAlchemy (0.9.8)
* redis (2.10.3)
* tldextract (1.5.1)
* BeautifulSoup4 (4.3.2)
